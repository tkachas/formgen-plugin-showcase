## Быстрый старт (3 команды)

```bash
npm install
npm run openapi-ts
npm run dev
```


Приложение с тремя формами:

1. форма сотрудника
2. форма компании
3. форма проекта


Генерация (npm run openapi-ts)

Плагин создает файлы в `src/client/`:

- `types.gen.ts` - TypeScript типы
- `forms.gen.ts` - схемы форм
